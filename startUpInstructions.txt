Posture Cam — Pi Zero 2 W startup commands
===========================================

ONE-TIME SETUP (after a fresh Pi OS flash)
------------------------------------------

sudo apt update
# Refresh the package index so apt knows what's installable.

sudo apt install -y rpicam-apps libopenblas-dev libgl1 v4l-utils cpufrequtils curl
# System libs: rpicam-vid (CSI camera), math/OpenCV runtime deps, governor tool, curl for uv.

curl -LsSf https://astral.sh/uv/install.sh | sh
# Installs uv, which we use to get Python 3.11 (MediaPipe has no 3.13 wheel).

source ~/.bashrc
# Puts uv on PATH for this shell.

uv python install 3.11
# Downloads a prebuilt Python 3.11 into uv's cache.

uv venv --python 3.11 ~/posture-venv
# Creates the project's virtual environment at ~/posture-venv using Python 3.11.

source ~/posture-venv/bin/activate
# Activates the venv (prompt should now start with "(posture-venv)").

pip install mediapipe opencv-python-headless pyserial flask numpy
# Installs all Python deps the script needs.

sudo systemctl disable --now bluetooth
# Turns off Bluetooth to free a little RAM/CPU and keep the Pi cooler.


EVERY BOOT
----------

sudo cpufreq-set -g ondemand
# Lets idle cores clock down between inferences (resets to default each reboot).

cd ~/Project
# Go to where laptopPostureCheck.py lives.

~/posture-venv/bin/python laptopPostureCheck.py
# Runs the script using the venv's Python directly; open http://<pi-ip>:5000/ in a browser to view.


QUITTING
--------

Ctrl-C
# Stops the script cleanly (releases camera and serial port).
